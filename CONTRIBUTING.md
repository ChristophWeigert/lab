# Contributing to Homelab Infrastructure

Thank you for your interest in contributing to this homelab infrastructure project! This is a personal homelab environment, but contributions are welcome and appreciated. Whether you're fixing a bug, adding a new service, or improving documentation, your help makes this project better.

This isn't enterprise software, but I do care about quality, maintainability, and following good practices.

## Code of Conduct

Be respectful, helpful, and patient:
- Welcome newcomers and first-time contributors
- Provide constructive feedback
- Assume good intentions
- Help others learn
- Keep discussions focused and professional

**First time contributing?** Welcome! Don't hesitate to ask questions. We were all beginners once, and we're happy to help you learn.

**Questions about this guide?** Open an issue and we'll clarify!
